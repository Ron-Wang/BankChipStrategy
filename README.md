## 🎯 策略逻辑

- **标的**：`515020.XSHG` 银行ETF + `588200.XSHG` 芯片ETF
- **初始权重**：银行60% / 芯片40%（固定不变）
- **调仓规则**：每季度最后一个交易日开盘，将持仓市值恢复至目标权重
- **交易成本**：佣金0.03%（双边），滑点0.1%，最低佣金5元
- **基准**：沪深300指数（`000300.XSHG`）
- **回测区间**：2022-12-29 至 2026-04-16（约3.3年）

### 核心理念

在A股市场，银行股代表低估值、高分红、防御属性，而芯片股代表高成长、高波动、国产替代主线。两者相关性较低，天然适合构成“哑铃策略”——一头稳，一头锐。季度再平衡本质上是一种低买高卖策略：当芯片大涨时自动减仓锁定利润，当芯片大跌时自动加仓摊低成本。

## 🚀 快速开始

### 前置条件

- 注册[聚宽](https://www.joinquant.com/)账号
- 了解Python基础语法

### 在聚宽平台运行

1. 登录聚宽平台，进入“我的策略”
2. 新建一个策略，将 `strategy.py` 中的代码完整粘贴
3. 设置回测参数：
   - 起始日期：`2022-01-01`（或两只ETF上市之后）
   - 初始资金：≥10,000元
   - 回测频率：日线
4. 点击“运行回测”
